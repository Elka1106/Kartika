# ‚òï Dasar Pemrograman Java di NetBeans

## üìñ Pengantar

Java adalah salah satu bahasa pemrograman yang **paling populer di dunia**, digunakan untuk membuat:

* Aplikasi desktop
* Aplikasi Android
* Aplikasi web
* Sistem backend (server)

Untuk menulis dan menjalankan program Java, kita bisa menggunakan **IDE NetBeans**, yang mempermudah proses coding, debugging, dan menjalankan proyek.

---

## üß© 1. Komponen Dasar Program Java di NetBeans

Setiap program Java memiliki **struktur dasar (komponen utama)** agar bisa dikenali oleh komputer.
Dalam NetBeans, struktur program Java biasanya terdiri dari:

### üß† a. Header (Bagian Atas Program)

Bagian **header** berisi:

* **Pernyataan paket (package)** ‚Äî menunjukkan lokasi file dalam proyek.
* **Import library** ‚Äî digunakan untuk menambahkan fungsi tambahan dari Java Library.

üìÑ **Contoh Header:**

```java
package myfirstproject;       // Menunjukkan nama paket
import java.util.Scanner;     // Mengimpor library untuk input data
```

> üí° `package` bersifat opsional jika proyek tidak menggunakan struktur folder kompleks.
> üí° `import` hanya digunakan jika kamu membutuhkan fitur tambahan seperti input, tanggal, atau database.

---

### ‚öôÔ∏è b. Body (Isi Utama Program)

Bagian **body** adalah bagian inti tempat logika program ditulis.
Biasanya terdiri dari:

1. **Deklarasi kelas (class)**
2. **Metode utama (main method)**
3. **Isi kode / logika program**

üìÑ **Contoh Body:**

```java
public class HelloWorld {             // Deklarasi kelas utama
    public static void main(String[] args) {    // Metode utama
        System.out.println("Hello, NetBeans!"); // Baris program (output)
    }
}
```

üß© **Penjelasan:**

* `public class HelloWorld` ‚Üí mendefinisikan sebuah kelas bernama *HelloWorld*.
* `public static void main(String[] args)` ‚Üí titik awal program Java dijalankan.
* `System.out.println()` ‚Üí perintah untuk menampilkan teks ke layar.

> ‚ö†Ô∏è Setiap program Java **harus memiliki** metode `main()` agar bisa dijalankan.

---

### üìò c. Footer (Bagian Penutup Program)

Bagian **footer** berisi tanda kurung kurawal penutup `{}` yang menandai akhir dari kelas atau metode.
Java sangat ketat terhadap pembukaan `{` dan penutupan `}`.

üìÑ **Contoh Footer:**

```java
    } // Menutup metode main
}     // Menutup kelas HelloWorld
```

> üí° Pastikan jumlah `{` dan `}` seimbang, jika tidak maka program akan error saat dijalankan.

---

## üî† 2. Sintaks Dasar Bahasa Pemrograman Java

### üß© a. Komentar (Comment)

Komentar digunakan untuk memberikan catatan di dalam kode, tidak dijalankan oleh komputer.

```java
// Komentar satu baris
/* Komentar
   lebih dari satu baris */
```

---

### üî¢ b. Variabel dan Tipe Data

Variabel digunakan untuk menyimpan nilai (angka, teks, dsb).

üìÑ **Contoh:**

```java
int umur = 18;              // Menyimpan angka bulat
double tinggi = 170.5;      // Menyimpan angka desimal
String nama = "Ayu";        // Menyimpan teks
boolean aktif = true;       // Menyimpan nilai benar/salah
```

üß† **Tipe Data Umum di Java:**

| Tipe Data | Deskripsi                | Contoh Nilai      |
| :-------- | :----------------------- | :---------------- |
| `int`     | Bilangan bulat           | `10`, `-5`, `100` |
| `double`  | Bilangan desimal         | `3.14`, `25.6`    |
| `char`    | Karakter tunggal         | `'A'`, `'b'`      |
| `String`  | Kumpulan karakter (teks) | `"Hello"`         |
| `boolean` | Nilai logika             | `true`, `false`   |

---

### üîÅ c. Operator Aritmatika

Operator digunakan untuk melakukan perhitungan.

| Operator | Fungsi      | Contoh  |
| :------- | :---------- | :------ |
| `+`      | Penjumlahan | `a + b` |
| `-`      | Pengurangan | `a - b` |
| `*`      | Perkalian   | `a * b` |
| `/`      | Pembagian   | `a / b` |
| `%`      | Sisa bagi   | `a % b` |

üìÑ **Contoh:**

```java
int a = 10, b = 3;
System.out.println(a + b); // Output: 13
System.out.println(a % b); // Output: 1
```

---

### üî£ d. Struktur Kontrol (Percabangan)

Digunakan untuk membuat keputusan.

üìÑ **Contoh If-Else:**

```java
int nilai = 80;

if (nilai >= 75) {
    System.out.println("Lulus");
} else {
    System.out.println("Tidak Lulus");
}
```

üìÑ **Contoh Switch:**

```java
int hari = 2;

switch (hari) {
    case 1: System.out.println("Senin"); break;
    case 2: System.out.println("Selasa"); break;
    default: System.out.println("Hari lain");
}
```

---

### üîÅ e. Perulangan (Loop)

Digunakan untuk mengulang perintah beberapa kali.

üìÑ **Contoh For Loop:**

```java
for (int i = 1; i <= 5; i++) {
    System.out.println("Perulangan ke-" + i);
}
```

üìÑ **Contoh While Loop:**

```java
int i = 1;
while (i <= 3) {
    System.out.println("Angka: " + i);
    i++;
}
```

---

### üì• f. Input dari Pengguna

Gunakan class `Scanner` untuk menerima input dari keyboard.

üìÑ **Contoh:**

```java
import java.util.Scanner;

public class InputContoh {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        System.out.print("Masukkan nama: ");
        String nama = input.nextLine();
        
        System.out.println("Halo, " + nama + "!");
    }
}
```

> üí° Jangan lupa menambahkan `import java.util.Scanner;` di bagian header untuk menggunakan fitur input.

---

### üì§ g. Output ke Layar

Gunakan perintah:

```java
System.out.println("Teks yang ingin ditampilkan");
```

* `println` = menampilkan teks dan ganti baris baru
* `print` = menampilkan teks tanpa pindah baris

---

## üß≠ Kesimpulan

Setiap program Java memiliki **tiga bagian utama:**

1. üß© **Header** ‚Üí deklarasi paket dan import library
2. ‚öôÔ∏è **Body** ‚Üí logika utama program (class dan method)
3. üìò **Footer** ‚Üí penutup kurung kurawal

Selain itu, kamu juga telah mempelajari **sintaks dasar Java**, seperti:

* Variabel dan tipe data
* Operator
* Percabangan dan perulangan
* Input dan output

> üí¨ ‚ÄúPahami struktur, kuasai sintaks ‚Äî dan kamu akan bisa membuat program apa pun.‚Äù

### 2. Latihan

Coba buat program Java baru di NetBeans dengan langkah berikut:

1. Tambahkan class bernama `Latihan2_Variabel`
2. Program : 
```Java
package latihan;

public class latihan_2_variable {
    public static  void main (String [] args){
        System.out.println ("=== PROGRAM BELAJAR VARIABLE JAVA ===");
        System.out.println();
        
        // 1. Varible primitive (Dasar)
        System.out.println ("1. VARIABEL PRIMITIVE");
        System.out.println("----------------------");
        
        // integer (Bilangan bulat)
        int umur = 20;
        byte nilaiUjian = 85;
        short jumlahSiswa = 150;
        long populasiKota = 5000000L;
        
        System.out. println("umur:" + umur);
        System.out.println ("Nilai Ujian:" + nilaiUjian);
        System.out.println("Jumlah Siswa: " + jumlahSiswa);
        System.out.println("Populasi Kota: " + populasiKota);
        System.out.println();
        
        //Floating point (bilangan desimal)
        float tinggiBadan = 175.5f;
        double hargaBarang = 125000.75;
        
        System.out.println("Tinggi Badan: " + tinggiBadan + " cm");
        System.out.println("Harga Barang: RP " + hargaBarang);
        System.out.println();
        
        // Chracter dan Boolean
        char grade = 'A' ;
        boolean isLulus = true;
        
        System.out.println("Grade: " + grade);
        System.out.println("Status Kelulusan: " + isLulus);
        System.out.println();
        
        // 2. Variabel Reference
        System.out.println("2. VARIABEL RERENCE");
        System.out.println("-------------------");
        
        String nama = "Budi Santoso";
        String alamat = "Jl. Merdeka No. 123";
        
        System.out.println("Nama: " + nama);
        System.out.println("Alamat: " + alamat);
        System.out.println();
        
        // 3. Konstanta (final)
        System.out.println ("3. KONSTANTA (FINAL)");
        System.out.println("---------------------");
        
        final double PI = 3.14159;
        final String NEGARA = "Indonesia";
        
        System.out.println ("Nilai PI: " + PI);
        System.out.println ("Negara: " + NEGARA);
        System.out.println();
        
        //4. Operasi dengan Variabel
        System.out.println("4. OPERASI DENGAN VARIABEL");
        System.out.println("--------------------------");
        
        int a = 10;
        int b = 5;
        
        int penjumlahan = a + b;
        int pengurangan = a - b;
        int perkalian = a * b;
        int pembagian = a / b;
        int modulus = a % b;
        
        System.out.println(a + " + " + b + " = " + penjumlahan);
        System.out.println(a + " - " + b + " = " + pengurangan);
        System.out.println(a + " * " + b + " = " + perkalian);
        System.out.println(a + " / " + b + " = " + pembagian);
        System.out.println(a + " % " + b + " = " + modulus);
        System.out.println();
        
        //5. Type Casting (konversi tipe data)
        System.out.println("5. TIPE CASTING");
        System.out.println("---------------");
        
        int angkaInt = 100;
        double angkaDoubel = angkaInt; // Widening casting (otomatis)
        int angkaKembali = (int) angkaDoubel; // Narrowing casting (manual)
        
        System.out.println("int ke double: " + angkaInt + " - " + angkaDoubel);
        System.out.println("double ke int: " + angkaDoubel + " - " + angkaKembali);
    }    
}
        

```
