## **Materi Lengkap: Pengenalan Linux & Cara Install Ubuntu di Oracle VM VirtualBox**

---

## **1. Pengenalan Linux**

### **Apa Itu Linux?**
Linux adalah **sistem operasi open-source** yang dikembangkan berdasarkan kernel Linux. Linux dikenal karena:
- **Gratis & Open Source**: Bisa dimodifikasi dan didistribusikan bebas.
- **Stabil & Aman**: Lebih tahan terhadap virus dan malware.
- **Ringan**: Bisa berjalan di hardware lama.
- **Kustomisasi Tinggi**: Banyak pilihan desktop environment (GNOME, KDE, XFCE, dll.).
- **Komunitas Besar**: Dukungan dari pengguna dan pengembang di seluruh dunia.

### **Distro Linux Populer**
- **Ubuntu**: Ramah pemula, banyak dukungan.
- **Linux Mint**: Mudah digunakan, mirip Windows.
- **Debian**: Stabil, cocok untuk server.
- **Fedora**: Terkini, sering digunakan pengembang.
- **Arch Linux**: Untuk pengguna tingkat lanjut.

---

## **2. Persiapan Install Ubuntu di VirtualBox**

### **Apa yang Dibutuhkan?**
- **Komputer** dengan spesifikasi minimal:
  - RAM: 2GB (rekomendasi 4GB)
  - Ruang disk: 25GB (rekomendasi 50GB)
  - CPU: Dual-core (rekomendasi quad-core)
- **Oracle VM VirtualBox**: [Unduh di sini](https://www.virtualbox.org/wiki/Downloads)
- **File ISO Ubuntu 22.04 LTS**: [Unduh di sini](https://ubuntu.com/download/desktop)

---

## **3. Langkah-langkah Install Ubuntu di VirtualBox**

### **A. Install Oracle VM VirtualBox**
1. Unduh VirtualBox sesuai sistem operasi kamu (Windows/macOS/Linux).
2. Jalankan installer dan ikuti petunjuk instalasi.
3. Buka VirtualBox setelah instalasi selesai.

---

### **B. Membuat Mesin Virtual Baru**
1. Klik **"New"** di VirtualBox.
2. Isi detail mesin virtual:
   - **Name**: Ubuntu 22.04
   - **Type**: Linux
   - **Version**: Ubuntu (64-bit)
3. Klik **"Next"**.

---

### **C. Alokasi RAM**
- Pilih jumlah RAM: **Minimal 2GB (rekomendasi 4GB)**.
- Klik **"Next"**.

---

### **D. Buat Virtual Hard Disk**
1. Pilih **"Create a virtual hard disk now"**.
2. Klik **"Create"**.
3. Pilih tipe hard disk: **VDI (VirtualBox Disk Image)**.
4. Pilih **"Dynamically allocated"** (ukuran disk akan bertambah sesuai kebutuhan).
5. Tentukan ukuran disk: **Minimal 25GB (rekomendasi 50GB)**.
6. Klik **"Create"**.

---

### **E. Konfigurasi Mesin Virtual**
1. Pilih mesin virtual Ubuntu yang baru dibuat.
2. Klik **"Settings"**.
3. **Tab "Storage"**:
   - Klik ikon CD di bawah "Controller: IDE".
   - Klik ikon CD di sebelah "Optical Drive".
   - Pilih "Choose a disk file" dan pilih file ISO Ubuntu yang sudah diunduh.
4. **Tab "System"**:
   - **Motherboard**: Centang "Enable EFI" (jika ingin menggunakan UEFI).
   - **Processor**: Beri minimal 2 CPU (jika komputer mendukung).
5. Klik **"OK"**.

---

### **F. Mulai Installasi Ubuntu**
1. Klik **"Start"** untuk menjalankan mesin virtual.
2. Pilih **"Try or Install Ubuntu"**.
3. Pilih bahasa: **"Bahasa Indonesia"** atau **"English"**.
4. Klik **"Install Ubuntu"**.

---

### **G. Proses Installasi Ubuntu**
1. Pilih layout keyboard: **"Indonesian"** atau **"English (US)"**.
2. Pilih jenis instalasi: **"Normal installation"** (rekomendasi).
3. Pilih **"Erase disk and install Ubuntu"** (tidak perlu khawatir, ini hanya virtual disk).
4. Konfirmasi partisi: Klik **"Continue"**.
5. Pilih zona waktu: **"Jakarta"**.
6. Isi data pengguna: Nama, nama komputer, username, dan password.
7. Tunggu proses instalasi selesai.
8. Klik **"Restart Now"** setelah instalasi selesai.

---

### **H. Setelah Installasi**
- **Login** dengan akun yang sudah dibuat.
- **Update sistem**:
  ```bash
  sudo apt update && sudo apt upgrade -y
  ```
- **Install VirtualBox Guest Additions** (untuk fitur tambahan seperti shared folder dan resolusi layar yang lebih baik):
  - Di menu VirtualBox, klik **Devices > Insert Guest Additions CD Image**.
  - Di Ubuntu, buka terminal dan jalankan:
    ```bash
    sudo apt install build-essential dkms linux-headers-$(uname -r)
    sudo mount /dev/cdrom /mnt
    cd /mnt
    sudo ./VBoxLinuxAdditions.run
    ```
  - Restart mesin virtual.

---

## **4. Tips Penggunaan VirtualBox**
- **Shared Folder**: Aktifkan di **Devices > Shared Folders** untuk berbagi file antara host dan guest.
- **Full Screen Mode**: Tekan **Host+F** (Host key default: `Right Ctrl`).
- **Snapshot**: Gunakan fitur snapshot untuk menyimpan keadaan mesin virtual sebelum melakukan perubahan penting.

---

## **5. Troubleshooting**
- **Jika Ubuntu tidak bisa boot**: Pastikan "Enable EFI" dicentang di Settings > System > Motherboard.
- **Jika layar terlalu kecil**: Install Guest Additions seperti langkah di atas.
