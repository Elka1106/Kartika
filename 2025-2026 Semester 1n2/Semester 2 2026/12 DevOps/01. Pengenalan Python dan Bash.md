## **Materi Dasar Python**

### 1. **Pengantar Python**
- Python adalah bahasa pemrograman tingkat tinggi, mudah dibaca, dan serbaguna.
- Digunakan untuk web development, data science, otomatisasi, dan banyak lagi.

### 2. **Instalasi Python**
- Unduh Python dari [python.org](https://www.python.org/downloads/)
- Pastikan Python sudah terpasang dengan menjalankan perintah di terminal:
  ```bash
  python --version
  ```

### 3. **Variabel dan Tipe Data**
- Variabel digunakan untuk menyimpan data.
- Tipe data dasar: `int`, `float`, `str`, `bool`, `list`, `tuple`, `dict`, `set`.

**Contoh:**
```python
nama = "Andi"  # string
umur = 25      # integer
tinggi = 170.5 # float
is_student = True # boolean
```

### 4. **Struktur Kontrol**
- **If-Else:**
  ```python
  if umur >= 18:
      print("Dewasa")
  else:
      print("Anak-anak")
  ```
- **Loop:**
  ```python
  for i in range(5):
      print(i)
  ```

### 5. **Fungsi**
- Fungsi digunakan untuk mengelompokkan kode yang dapat digunakan kembali.
  ```python
  def sapaan(nama):
      return f"Halo, {nama}!"

  print(sapaan("Budi"))
  ```

### 6. **List dan Dictionary**
- **List:**
  ```python
  buah = ["apel", "pisang", "jeruk"]
  ```
- **Dictionary:**
  ```python
  orang = {"nama": "Andi", "umur": 25}
  ```

### 7. **Input/Output**
- **Input:**
  ```python
  nama = input("Masukkan nama: ")
  ```
- **Output:**
  ```python
  print(f"Halo, {nama}!")
  ```

## **Materi Dasar Bash**

### 1. **Pengantar Bash**
- Bash (Bourne Again SHell) adalah shell dan bahasa scripting untuk sistem operasi Unix/Linux.
- Digunakan untuk otomatisasi tugas, manajemen file, dan administrasi sistem.

### 2. **Perintah Dasar Bash**


| Perintah         | Penjelasan                                                                 | Contoh Penggunaan                     |
|------------------|----------------------------------------------------------------------------|---------------------------------------|
| **ls**           | Menampilkan daftar file dan direktori                                      | `ls -l` (tampilkan detail)            |
| **cd**           | Berpindah direktori                                                        | `cd /home/user`                       |
| **pwd**          | Menampilkan direktori kerja saat ini                                       | `pwd`                                 |
| **mkdir**        | Membuat direktori baru                                                     | `mkdir folder_baru`                   |
| **rmdir**        | Menghapus direktori kosong                                                 | `rmdir folder_kosong`                 |
| **rm**           | Menghapus file/direktori                                                   | `rm file.txt`                         |
| **cp**           | Menyalin file/direktori                                                    | `cp file.txt /backup/`                |
| **mv**           | Memindahkan atau mengganti nama file/direktori                             | `mv lama.txt baru.txt`                |
| **touch**        | Membuat file kosong atau memperbarui timestamp file                         | `touch file_baru.txt`                 |
| **cat**          | Menampilkan isi file                                                       | `cat file.txt`                        |
| **echo**         | Menampilkan teks atau variabel                                             | `echo "Halo"`                         |
| **grep**         | Mencari teks dalam file                                                    | `grep "kata" file.txt`                |
| **find**         | Mencari file/direktori                                                     | `find / -name "file.txt"`             |
| **chmod**        | Mengubah izin (permission) file/direktori                                 | `chmod 755 script.sh`                 |
| **chown**        | Mengubah kepemilikan file/direktori                                        | `chown user:group file.txt`           |
| **ps**           | Menampilkan proses yang sedang berjalan                                    | `ps aux`                              |
| **kill**         | Menghentikan proses berdasarkan PID                                        | `kill 1234`                           |
| **top/htop**     | Menampilkan penggunaan sumber daya sistem                                 | `top`                                 |
| **df**           | Menampilkan penggunaan ruang disk                                          | `df -h`                               |
| **du**           | Menampilkan penggunaan ruang oleh file/direktori                           | `du -sh /home`                        |
| **tar**          | Mengompresi atau mengekstrak file                                          | `tar -xzvf arsip.tar.gz`              |
| **ssh**          | Terhubung ke server remote                                                 | `ssh user@host`                       |
| **scp**          | Menyalin file secara aman antar host                                       | `scp file.txt user@host:/path/`        |
| **wget**         | Mengunduh file dari internet                                               | `wget https://example.com/file.zip`   |
| **curl**         | Mengirim/menerima data dari server (HTTP, FTP, dll.)                       | `curl -O https://example.com/file.zip`|
| **man**          | Menampilkan manual (bantuan) perintah                                     | `man ls`                              |
| **alias**        | Membuat alias untuk perintah                                               | `alias ll='ls -la'`                   |
| **history**      | Menampilkan riwayat perintah                                              | `history`                             |
| **cron**         | Menjadwalkan tugas otomatis                                                | `crontab -e`                          |
| **export**       | Menetapkan variabel lingkungan                                             | `export PATH=$PATH:/new/path`         |
| **source**       | Menjalankan script di shell saat ini                                       | `source ~/.bashrc`                    |
| **ifconfig**     | Menampilkan informasi jaringan (Linux)                                     | `ifconfig`                            |
| **ping**         | Memeriksa koneksi jaringan                                                 | `ping google.com`                     |
| **sudo**         | Menjalankan perintah sebagai superuser                                     | `sudo apt update`                     |
| **apt/yum**      | Manajemen paket (Debian/Ubuntu: apt, CentOS/RHEL: yum)                      | `sudo apt install nano`               |
| **nano/vi/vim**  | Editor teks berbasis terminal                                              | `nano file.txt`                       |
| **exit**         | Keluar dari shell                                                          | `exit`                                |

---
**Catatan:**
- Gunakan `man [perintah]` untuk melihat dokumentasi lengkap.
- Beberapa perintah memerlukan hak admin (gunakan `sudo`).

### 3. **Variabel di Bash**
```bash
nama="Andi"
echo "Halo, $nama!"
```

### 4. **Struktur Kontrol**
- **If-Else:**
  ```bash
  if [ $umur -ge 18 ]; then
      echo "Dewasa"
  else
      echo "Anak-anak"
  fi
  ```
- **Loop:**
  ```bash
  for i in {1..5}; do
      echo $i
  done
  ```

### 5. **Script Bash**
- Simpan script di file, misal `script.sh`:
  ```bash
  #!/bin/bash
  echo "Halo, dunia!"
  ```
- Jalankan script:
  ```bash
  chmod +x script.sh
  ./script.sh
  ```
