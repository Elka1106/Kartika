## **Materi Dasar Python**

### 1. **Pengantar Python**
- Python adalah bahasa pemrograman tingkat tinggi, mudah dibaca, dan serbaguna.
- Digunakan untuk web development, data science, otomatisasi, dan banyak lagi.

### 2. **Instalasi Python**
- Unduh Python dari [python.org](https://www.python.org/downloads/)
- Pastikan Python sudah terpasang dengan menjalankan perintah di terminal:
  ```bash
  python --version
  ```

### 3. **Variabel dan Tipe Data**
- Variabel digunakan untuk menyimpan data.
- Tipe data dasar: `int`, `float`, `str`, `bool`, `list`, `tuple`, `dict`, `set`.

**Contoh:**
```python
nama = "Andi"  # string
umur = 25      # integer
tinggi = 170.5 # float
is_student = True # boolean
```

### 4. **Struktur Kontrol**
- **If-Else:**
  ```python
  if umur >= 18:
      print("Dewasa")
  else:
      print("Anak-anak")
  ```
- **Loop:**
  ```python
  for i in range(5):
      print(i)
  ```

### 5. **Fungsi**
- Fungsi digunakan untuk mengelompokkan kode yang dapat digunakan kembali.
  ```python
  def sapaan(nama):
      return f"Halo, {nama}!"

  print(sapaan("Budi"))
  ```

### 6. **List dan Dictionary**
- **List:**
  ```python
  buah = ["apel", "pisang", "jeruk"]
  ```
- **Dictionary:**
  ```python
  orang = {"nama": "Andi", "umur": 25}
  ```

### 7. **Input/Output**
- **Input:**
  ```python
  nama = input("Masukkan nama: ")
  ```
- **Output:**
  ```python
  print(f"Halo, {nama}!")
  ```

### **Penjelasan `chmod` di Bash**

**`chmod`** (change mode) adalah perintah di Bash yang digunakan untuk **mengubah izin (permission) file atau direktori** di sistem operasi Linux/Unix. Izin ini menentukan siapa yang boleh membaca, menulis, atau menjalankan file/direktori.

---

## **1. Jenis Izin (Permission) di Linux**

Ada tiga jenis izin utama:
- **Read (r):** Izin untuk membaca isi file/direktori.
- **Write (w):** Izin untuk mengubah isi file/direktori.
- **Execute (x):** Izin untuk menjalankan file (jika file adalah program/script) atau masuk ke direktori.

Dan tiga kategori pengguna:
- **Owner (u):** Pemilik file/direktori.
- **Group (g):** Kelompok pengguna yang memiliki akses.
- **Others (o):** Semua pengguna lain.

---

## **2. Format Perintah `chmod`**

### **A. Menggunakan Simbol (u/g/o/a + r/w/x + +/-)**
```bash
chmod [siapa][operator][izin] file/direktori
```
- **Siapa:** `u` (user/owner), `g` (group), `o` (others), `a` (all)
- **Operator:** `+` (tambah izin), `-` (hapus izin), `=` (set izin)
- **Izin:** `r` (read), `w` (write), `x` (execute)

**Contoh:**
```bash
chmod u+x script.sh   # Beri izin execute untuk owner
chmod g-w file.txt    # Hapus izin write untuk group
chmod o=r file.txt    # Set izin read-only untuk others
chmod a+r file.txt    # Beri izin read untuk semua
```

---

### **B. Menggunakan Angka (Octal)**
Setiap izin memiliki nilai angka:
- `r` = 4
- `w` = 2
- `x` = 1

Jumlahkan nilai untuk setiap kategori (owner, group, others).

**Contoh:**
```bash
chmod 755 script.sh
```
- **7** (4+2+1) = `rwx` untuk owner
- **5** (4+0+1) = `r-x` untuk group
- **5** (4+0+1) = `r-x` untuk others

**Contoh lain:**
- `chmod 644 file.txt` → `rw-r--r--`
- `chmod 700 file.txt` → `rwx------`

---

## **3. Contoh Penggunaan Umum**

| Perintah                | Keterangan                                      |
|-------------------------|-------------------------------------------------|
| `chmod +x script.sh`    | Beri izin execute untuk semua                  |
| `chmod 755 script.sh`   | Beri izin rwx untuk owner, rx untuk group/others |
| `chmod 644 file.txt`    | Beri izin rw untuk owner, r untuk group/others |
| `chmod u=rw,go=r file.txt` | Set izin rw untuk owner, r untuk group/others |
| `chmod -R 755 folder/`  | Ubah izin secara rekursif untuk semua file/direktori di dalam folder |

---

## **4. Tips Penting**
- Gunakan `ls -l` untuk melihat izin file/direktori saat ini.
- Izin direktori harus memiliki `x` agar bisa diakses.
- Hati-hati dengan `chmod -R`, karena akan mengubah izin untuk semua file/direktori di dalamnya.

---
**Contoh Output `ls -l`:**
```
-rwxr-xr-x 1 user group 1024 Jan 5 10:00 script.sh
```
- `-rwxr-xr-x` = izin untuk owner, group, others.
