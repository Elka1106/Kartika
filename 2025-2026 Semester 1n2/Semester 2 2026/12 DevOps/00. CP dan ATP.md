**Capaian Pembelajaran (CP) DevOps Kelas 12 SMK:**

Pada akhir pembelajaran, peserta didik mampu memahami konsep dasar DevOps sebagai metode kolaborasi antara pengembang dan tim operasi, serta menguasai keterampilan praktis seperti penggunaan sistem operasi Linux, dasar pemrograman skrip (Bash/Python), cloud (AWS/Azure/GCP), pengenalan kontainerisasi dengan Docker, otomatisasi sederhana menggunakan CI/CD (seperti GitHub Actions), dan pengelolaan infrastruktur dasar.

### **Alur Tujuan Pembelajaran (ATP) DevOps Kelas 12 SMK**
**Semester: 1**
**Durasi: 18 Minggu (1 Semester)**

---

#### **Tujuan Umum:**
Peserta didik mampu memahami konsep dasar DevOps, menguasai keterampilan teknis (Linux, Bash/Python, cloud, Docker, CI/CD), dan menerapkannya dalam proyek sederhana.

---

### **ATP DevOps Kelas 12 SMK**

| **Minggu** | **Tujuan Pembelajaran**                                                                 | **Kegiatan Pembelajaran**                                                                                     | **Penilaian**                                      |
|------------|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|----------------------------------------------------|
| **1-2**    | **Memahami Konsep Dasar DevOps**                                                       | Diskusi kelompok, studi kasus, dan presentasi tentang prinsip DevOps dan manfaatnya dalam industri.         | Tugas presentasi, kuis.                            |
| **3-4**    | **Menggunakan Sistem Operasi Linux**                                                   | Latihan command line Linux, manajemen file, dan pengaturan izin dasar.                                      | Tugas praktik, ujian praktik.                     |
| **5-6**    | **Menerapkan Dasar Pemrograman Skrip (Bash/Python)**                                   | Latihan menulis skrip Bash/Python untuk otomatisasi tugas sehari-hari.                                       | Tugas praktik, penilaian skrip.                   |
| **7-8**    | **Mengenal Layanan Cloud (AWS/Azure/GCP)**                                            | Latihan membuat akun cloud gratis, membuat instance virtual, dan mengaksesnya via SSH.                      | Tugas proyek, penilaian konfigurasi cloud.        |
| **9-10**   | **Memahami dan Menggunakan Docker untuk Kontainerisasi**                              | Latihan membuat Dockerfile, membangun image, dan menjalankan container.                                     | Tugas praktik, ujian praktik.                     |
| **11-12**  | **Menerapkan Otomatisasi Sederhana dengan CI/CD (GitHub Actions)**                     | Latihan membuat workflow GitHub Actions untuk otomatisasi pengujian dan penyebaran sederhana.               | Tugas proyek, penilaian pipeline.                 |
| **13-14**  | **Mengelola Infrastruktur Dasar (Ansible/Manual)**                                    | Latihan menulis playbook Ansible untuk mengotomatiskan tugas konfigurasi server.                            | Tugas proyek, penilaian skrip.                    |
| **15-16**  | **Mengembangkan Soft Skill: Kolaborasi, Komunikasi, dan Pemecahan Masalah**            | Simulasi kerja tim, presentasi proyek, dan studi kasus pemecahan masalah.                                  | Penilaian presentasi, refleksi kelompok.         |
| **17-18**  | **Membangun dan Mempresentasikan Portofolio Proyek DevOps Sederhana**                  | Peserta didik merancang, mengimplementasikan, dan mendokumentasikan proyek DevOps sederhana.               | Penilaian portofolio, presentasi proyek.         |
