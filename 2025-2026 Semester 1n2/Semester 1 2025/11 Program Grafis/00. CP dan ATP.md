CP :
Peserta didik mampu menerapkan perintah eksekusi bahasa pemrograman berbasis teks, grafik, dan multimedia yang mengimplementasikan pemrograman terstruktur, pemrograman berorientasi objek lanjutan, pemodelan perangkat lunak, pemrograman antarmuka GUI (Graphical User Interface), dan pemanfaatan pustaka (library).


---

## **Alur Tujuan Pembelajaran (ATP) Kelas 11**
**Mata Pelajaran:** Pemrograman
**Alokasi Waktu:** 32 minggu (2 semester)

---

## **Semester 1: Dasar-Dasar Pemrograman dan Pemrograman Terstruktur**

### **Tujuan Pembelajaran Umum Semester 1**
Peserta didik mampu memahami dan menerapkan konsep dasar pemrograman, pemrograman terstruktur, serta mengembangkan program sederhana dengan output teks dan grafik.

---

### **Alur Pembelajaran per Minggu (Semester 1)**

| **Minggu** | **Tujuan Pembelajaran Khusus** | **Kegiatan Pembelajaran** | **Penilaian** |
|------------|-------------------------------|---------------------------|---------------|
| **1-2** | Memahami konsep dasar algoritma dan pemrograman | - Pengenalan algoritma dan flowchart <br> - Pengenalan bahasa pemrograman (misal: Python, Java) | Tugas: membuat algoritma sederhana dan flowchart |
| **3-4** | Menerapkan struktur dasar pemrograman | - Variabel, tipe data, operator <br> - Struktur kontrol (if, else, loop) | Praktik: menulis program sederhana dengan struktur kontrol |
| **5-6** | Menerapkan fungsi dan prosedur | - Pengenalan fungsi dan prosedur <br> - Praktik membuat program modular | Tugas: membuat program dengan fungsi/prosedur |
| **7-8** | Menerapkan array dan struktur data sederhana | - Pengenalan array 1D dan 2D <br> - Praktik pengolahan data menggunakan array | Praktik: program pengolahan data dengan array |
| **9-10** | Menerapkan pemrograman berbasis teks dan grafik sederhana | - Input/output teks <br> - Pengenalan library grafik sederhana (misal: Turtle untuk Python) | Proyek mini: program dengan output teks dan grafik |
| **11-12** | Menerapkan pemrograman terstruktur | - Modularitas, fungsi, dan prosedur <br> - Praktik membuat program terstruktur | Tugas: program terstruktur dengan fungsi/prosedur |
| **13-14** | Pengenalan pemrograman berorientasi objek dasar | - Class, object, attribute, method <br> - Praktik membuat class sederhana | Praktik: membuat program dengan 1-2 class |
| **15-16** | Evaluasi dan proyek akhir semester 1 | - Ujian praktik <br> - Proyek: program sederhana dengan pemrograman terstruktur | Penilaian: ujian praktik dan proyek |

---

## **Semester 2: Pemrograman Berorientasi Objek, GUI, dan Multimedia**

### **Tujuan Pembelajaran Umum Semester 2**
Peserta didik mampu menerapkan pemrograman berorientasi objek lanjutan, pemodelan perangkat lunak, antarmuka GUI, dan pemanfaatan pustaka untuk mengembangkan aplikasi sederhana dengan elemen multimedia.

---

### **Alur Pembelajaran per Minggu (Semester 2)**

| **Minggu** | **Tujuan Pembelajaran Khusus** | **Kegiatan Pembelajaran** | **Penilaian** |
|------------|-------------------------------|---------------------------|---------------|
| **17-18** | Menerapkan OOP lanjutan | - Inheritance, polymorphism, encapsulation <br> - Praktik membuat program dengan OOP | Tugas: program dengan inheritance dan polymorphism |
| **19-20** | Menerapkan pemodelan perangkat lunak | - Pengenalan UML (Use Case, Class Diagram) <br> - Praktik membuat diagram untuk kasus sederhana | Tugas: membuat diagram UML untuk studi kasus |
| **21-22** | Menerapkan pemrograman antarmuka GUI | - Pengenalan library GUI (misal: Tkinter, PyQt) <br> - Praktik membuat form sederhana | Praktik: membuat aplikasi GUI dengan fitur dasar |
| **23-24** | Menerapkan pemrograman multimedia | - Pengenalan library multimedia (misal: Pillow, Pygame) <br> - Integrasi gambar, suara, animasi sederhana | Proyek mini: program dengan elemen multimedia |
| **25-26** | Menerapkan pemanfaatan pustaka (library) | - Pengenalan dan instalasi library eksternal <br> - Praktik menggunakan library untuk memperkaya fungsionalitas | Tugas: program dengan minimal 2 library eksternal |
| **27-28** | Integrasi semua konsep dalam proyek akhir | - Pengembangan aplikasi sederhana (misal: manajemen perpustakaan, toko online) <br> - Presentasi dan uji coba program | Proyek akhir: aplikasi dengan GUI, multimedia, dan dokumentasi UML |
| **29-30** | Evaluasi dan presentasi proyek akhir | - Presentasi proyek <br> - Evaluasi dan refleksi pembelajaran | Penilaian: presentasi dan laporan proyek |
| **31-32** | Ujian akhir semester 2 | - Ujian teori dan praktik <br> - Review materi selama 1 tahun | Penilaian: ujian akhir |

---

### **Capaian per Fase Pembelajaran**
- **Semester 1:** Peserta didik mampu membuat program sederhana dengan pemrograman terstruktur, fungsi, array, dan output teks/grafik.
- **Semester 2:** Peserta didik mampu membuat aplikasi sederhana dengan OOP lanjutan, GUI, multimedia, dan dokumentasi UML.

---

### **Penilaian**
- **Penilaian Harian:** Tugas praktik, kuis, partisipasi (30%)
- **Penilaian Tengah Semester:** Ujian praktik dan proyek mini (30%)
- **Penilaian Akhir Semester:** Proyek akhir dan presentasi (40%)
