
## üèóÔ∏è **1. CLASS DAN OBJECT - PENJELASAN SUPER DETAIL**

### **üìö DEFINISI:**
- **CLASS**: Template atau cetakan untuk membuat object. Class hanya berisi deklarasi (rancangan) tanpa nilai nyata.
- **OBJECT**: Instance atau wujud nyata dari class. Object memiliki nilai konkret dan bisa melakukan aksi.

### **üîß ANALOGI:**
- **Class** = Formulir pendaftaran kosong
- **Object** = Formulir yang sudah diisi dengan data lengkap

### **üíª CONTOH KODE LENGKAP:**

```java
// üìÑ CLASS: Template untuk membuat mahasiswa
public class Mahasiswa {
    // ==================== BAGIAN 1: ATRIBUT ====================
    // Atribut adalah data yang disimpan dalam object
    public String nama;      // Menyimpan nama mahasiswa
    public String nim;       // Menyimpan nomor induk mahasiswa  
    public String jurusan;   // Menyimpan jurusan
    public int umur;        // Menyimpan umur
    public double ipk;      // Menyimpan IPK
    
    // ==================== BAGIAN 2: METHOD ====================
    // Method adalah aksi atau perilaku yang bisa dilakukan object
    
    // Method tanpa parameter
    public void perkenalanDiri() {
        System.out.println("Halo, nama saya " + nama);
        System.out.println("NIM: " + nim + ", Jurusan: " + jurusan);
    }
    
    // Method dengan parameter
    public void belajar(String mataKuliah, int jamBelajar) {
        System.out.println(nama + " sedang belajar " + mataKuliah);
        System.out.println("Durasi: " + jamBelajar + " jam");
    }
    
    // Method dengan return value
    public String dapatkanInfo() {
        String info = "Nama: " + nama + ", NIM: " + nim + ", IPK: " + ipk;
        return info;
    }
    
    // Method untuk menampilkan semua data
    public void tampilkanDataLengkap() {
        System.out.println("=== DATA MAHASISWA ===");
        System.out.println("Nama    : " + nama);
        System.out.println("NIM     : " + nim);
        System.out.println("Jurusan : " + jurusan);
        System.out.println("Umur    : " + umur + " tahun");
        System.out.println("IPK     : " + ipk);
        System.out.println("======================");
    }
}

// üéØ PROGRAM UTAMA: Tempat membuat dan menggunakan object
public class ProgramKampus {
    public static void main(String[] args) {
        
        System.out.println("üéì PROGRAM MANAJEMEN MAHASISWA");
        System.out.println("==============================\n");
        
        // ==================== MEMBUAT OBJECT ====================
        System.out.println("1. MEMBUAT OBJECT MAHASISWA");
        
        // Object 1: Mahasiswa pertama
        Mahasiswa mahasiswa1 = new Mahasiswa();
        
        // Mengisi data untuk object mahasiswa1
        mahasiswa1.nama = "Budi Santoso";
        mahasiswa1.nim = "20230001";
        mahasiswa1.jurusan = "Teknik Informatika";
        mahasiswa1.umur = 20;
        mahasiswa1.ipk = 3.75;
        
        // Object 2: Mahasiswa kedua  
        Mahasiswa mahasiswa2 = new Mahasiswa();
        mahasiswa2.nama = "Sari Dewi";
        mahasiswa2.nim = "20230002";
        mahasiswa2.jurusan = "Sistem Informasi";
        mahasiswa2.umur = 19;
        mahasiswa2.ipk = 3.89;
        
        // Object 3: Mahasiswa ketiga
        Mahasiswa mahasiswa3 = new Mahasiswa();
        mahasiswa3.nama = "Andi Pratama";
        mahasiswa3.nim = "20230003";
        mahasiswa3.jurusan = "Teknik Elektro";
        mahasiswa3.umur = 21;
        mahasiswa3.ipk = 3.25;
        
        // ==================== MENGGUNAKAN OBJECT ====================
        System.out.println("\n2. MENGGUNAKAN METHOD PADA OBJECT");
        
        System.out.println("\n--- Mahasiswa 1 ---");
        mahasiswa1.perkenalanDiri();
        mahasiswa1.belajar("Pemrograman Java", 3);
        mahasiswa1.tampilkanDataLengkap();
        
        System.out.println("\n--- Mahasiswa 2 ---");
        mahasiswa2.perkenalanDiri();
        mahasiswa2.belajar("Basis Data", 2);
        mahasiswa2.tampilkanDataLengkap();
        
        System.out.println("\n--- Mahasiswa 3 ---");
        mahasiswa3.perkenalanDiri();
        mahasiswa3.belajar("Matematika Diskrit", 4);
        mahasiswa3.tampilkanDataLengkap();
        
        // ==================== OPERASI LAINNYA ====================
        System.out.println("\n3. OPERASI LAIN PADA OBJECT");
        
        // Menggunakan return value dari method
        String infoBudi = mahasiswa1.dapatkanInfo();
        System.out.println("Info Budi: " + infoBudi);
        
        // Mengubah nilai atribut
        System.out.println("\n4. MENGUBAH DATA OBJECT");
        System.out.println("IPK " + mahasiswa1.nama + " sebelum: " + mahasiswa1.ipk);
        mahasiswa1.ipk = 3.95;  // IPK naik
        System.out.println("IPK " + mahasiswa1.nama + " sesudah: " + mahasiswa1.ipk);
        
        // Membandingkan object
        System.out.println("\n5. MEMBANDINGKAN OBJECT");
        System.out.println("mahasiswa1 == mahasiswa2? " + (mahasiswa1 == mahasiswa2));
        
        // Menghitung total IPK
        System.out.println("\n6. STATISTIK MAHASISWA");
        double totalIpk = mahasiswa1.ipk + mahasiswa2.ipk + mahasiswa3.ipk;
        double rataRataIpk = totalIpk / 3;
        System.out.println("Rata-rata IPK: " + rataRataIpk);
        
        System.out.println("\n‚ú® PROGRAM SELESAI");
    }
}
```

### **üéØ POIN-PENTING YANG DIPELAJARI:**

1. **Cara membuat class** dengan atribut dan method
2. **Cara membuat object** dengan keyword `new`
3. **Cara mengisi data** ke dalam object
4. **Cara memanggil method** pada object
5. **Cara mengubah nilai** atribut object
6. **Cara menggunakan return value** dari method
7. **Setiap object independen** - perubahan di satu object tidak mempengaruhi object lain

### **üí° KESIMPULAN:**
- **1 Class** bisa digunakan untuk membuat **banyak Object**
- Setiap **Object** memiliki data yang berbeda-beda
- **Object** bisa melakukan aksi melalui **method**
- **Class** hanyalah template, **Object** adalah realitas
