## 1. Praktik 1: Membuat Form GUI Sederhana (Hello GUI)

**Tujuan**
Peserta didik mampu membuat tampilan GUI sederhana menggunakan NetBeans.

**Langkah Praktik**

* Buat project **Java â†’ Java Application**
* Buat **JFrame Form**
* Tambahkan:

  * `JLabel`
  * `JButton`

Code :
'''
import javax.swing.*;

public class HelloGUI extends JFrame {

    public HelloGUI() {
        // Set judul frame
        setTitle("Hello NetBeans GUI");

        // Set ukuran frame
        setSize(300, 200);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLocationRelativeTo(null); // Agar frame muncul di tengah layar

        // Buat JLabel
        JLabel label = new JLabel("Hello NetBeans GUI");
        label.setBounds(50, 50, 200, 30); // (x, y, width, height)

        // Buat JButton
        JButton button = new JButton("Klik Saya");
        button.setBounds(50, 100, 200, 30);

        // Tambahkan komponen ke frame
        add(label);
        add(button);

        // Set layout menjadi null agar bisa atur posisi manual
        setLayout(null);
    }

    public static void main(String[] args) {
        // Buat objek HelloGUI dan tampilkan
        HelloGUI frame = new HelloGUI();
        frame.setVisible(true);
    }
}
'''

**Tugas**

* Ubah teks `JLabel` menjadi **â€œHello NetBeans GUIâ€**
* Atur ukuran dan posisi komponen

**Hasil yang Diharapkan**

* Aplikasi GUI muncul dengan label dan tombol

## 2. Praktik 2: Tombol Interaktif (Button Action)

**Tujuan**
Peserta didik mampu menggunakan event pada tombol.

**Komponen**

* `JLabel`
* `JButton`

**Code :**
'''
import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class ButtonActionGUI extends JFrame {

    public ButtonActionGUI() {
        // Set judul frame
        setTitle("Tombol Interaktif");

        // Set ukuran frame
        setSize(300, 200);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLocationRelativeTo(null); // Agar frame muncul di tengah layar

        // Buat JLabel
        JLabel label = new JLabel("Halo, klik tombol di bawah!");
        label.setBounds(50, 50, 250, 30);

        // Buat JButton
        JButton button = new JButton("Klik Saya");
        button.setBounds(50, 100, 200, 30);

        // Tambahkan ActionListener pada tombol
        button.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                label.setText("Tombol sudah diklik!");
            }
        });

        // Tambahkan komponen ke frame
        add(label);
        add(button);

        // Set layout menjadi null agar bisa atur posisi manual
        setLayout(null);
    }

    public static void main(String[] args) {
        // Buat objek ButtonActionGUI dan tampilkan
        ButtonActionGUI frame = new ButtonActionGUI();
        frame.setVisible(true);
    }
}

'''

**Tugas**

* Ketika tombol diklik, teks `JLabel` berubah menjadi
  **â€œTombol Berhasil Diklikâ€**

**Konsep yang Dipelajari**

* Event `ActionPerformed`
* Interaksi pengguna
