## **1. Apa Itu Python?**
**Python** adalah bahasa pemrograman yang dirancang agar mudah dipahami dan digunakan, baik oleh pemula maupun programmer profesional. Sebagai bahasa **tingkat tinggi**, Python memungkinkan kita menulis kode dengan sintaksis sederhana dan mirip bahasa sehari-hari, sehingga tidak perlu memikirkan detail teknis yang rumit seperti pada bahasa pemrograman tingkat rendah. Python juga termasuk bahasa **interpreted**, artinya kode yang kita tulis dapat langsung dijalankan baris per baris tanpa perlu dikompilasi terlebih dahulu, sehingga proses pengembangan menjadi lebih cepat dan fleksibel. Selain itu, Python mendukung **multi-paradigma**, yang berarti kita bisa menulis program dengan berbagai gaya, seperti pemrograman prosedural (langkah demi langkah), pemrograman berorientasi objek (OOP) untuk membuat struktur kode yang terorganisir, maupun pemrograman fungsional yang fokus pada penggunaan fungsi-fungsi kecil. Dengan keunggulan ini, Python cocok digunakan untuk berbagai keperluan, mulai dari membuat website, menganalisis data, hingga mengembangkan kecerdasan buatan. Python dikenal karena sintaksisnya yang **sederhana**, **mudah dibaca**, dan **serbaguna**. Python banyak digunakan untuk:
- Pengembangan web (Django, Flask)
- Data Science & AI (NumPy, Pandas, TensorFlow)
- Otomatisasi & scripting
- Game & aplikasi desktop

### 1. **Integer (int)**
Digunakan untuk menyimpan bilangan bulat (positif/negatif).
**Contoh:**
```python
umur = 20
print(umur)  # Output: 20
print(type(umur))  # Output: <class 'int'>
```

---

### 2. **Float (float)**
Digunakan untuk menyimpan bilangan desimal (pecahan).
**Contoh:**
```python
tinggi = 175.5
print(tinggi)  # Output: 175.5
print(type(tinggi))  # Output: <class 'float'>
```

---

### 3. **String (str)**
Digunakan untuk menyimpan teks atau karakter, diawali dan diakhiri dengan tanda petik (`'` atau `"`).
**Contoh:**
```python
nama = "Budi"
print(nama)  # Output: Budi
print(type(nama))  # Output: <class 'str'>
```

---

### 4. **Boolean (bool)**
Hanya memiliki dua nilai: `True` atau `False`.
**Contoh:**
```python
is_student = True
print(is_student)  # Output: True
print(type(is_student))  # Output: <class 'bool'>
```

---


### 1. Gerbang AND Sederhana
```python
a, b = True, False
print(a and b)  # Output: False
```
**Penjelasan:**
Gerbang AND hanya menghasilkan `True` jika **kedua input bernilai `True`**. Di sini, `a` benar (`True`), `b` salah (`False`), jadi hasilnya `False`.

---

### 2. Gerbang OR Sederhana
```python
a, b = True, False
print(a or b)  # Output: True
```
**Penjelasan:**
Gerbang OR menghasilkan `True` jika **setidaknya satu input bernilai `True`**. Karena `a` benar, hasilnya `True`.

---

### 3. Gerbang NOT
```python
a = True
print(not a)  # Output: False
```
**Penjelasan:**
Gerbang NOT **membalik nilai input**. Jika input `True`, output `False`, dan sebaliknya.

---

### 4. Gerbang NAND
```python
a, b = True, True
print(not (a and b))  # Output: False
```
**Penjelasan:**
NAND adalah kebalikan dari AND. Jika AND menghasilkan `True`, NAND menghasilkan `False`. Di sini, `a` dan `b` keduanya `True`, jadi AND-nya `True`, dan NAND-nya `False`.

---

### 5. Gerbang NOR
```python
a, b = True, False
print(not (a or b))  # Output: False
```
**Penjelasan:**
NOR adalah kebalikan dari OR. Jika OR menghasilkan `True`, NOR menghasilkan `False`. Di sini, OR-nya `True`, jadi NOR-nya `False`.

---

### 6. Gerbang XOR
```python
a, b = True, False
print(a != b)  # Output: True
```
**Penjelasan:**
XOR menghasilkan `True` jika **kedua input berbeda**. `a` benar, `b` salah, jadi hasilnya `True`.

---

### 7. Gerbang XNOR
```python
a, b = True, False
print(a == b)  # Output: False
```
**Penjelasan:**
XNOR menghasilkan `True` jika **kedua input sama**. Di sini, `a` dan `b` berbeda, jadi hasilnya `False`.

---

### 8. Gerbang AND dengan 3 Input
```python
a, b, c = True, True, False
print(a and b and c)  # Output: False
```
**Penjelasan:**
AND dengan 3 input hanya `True` jika **semua input `True`**. Karena `c` `False`, hasilnya `False`.

---

### 9. Gerbang OR dengan 3 Input
```python
a, b, c = True, False, False
print(a or b or c)  # Output: True
```
**Penjelasan:**
OR dengan 3 input `True` jika **setidaknya satu input `True`**. Karena `a` `True`, hasilnya `True`.

---

### 10. Kombinasi AND + OR
```python
a, b, c = True, False, True
print((a and b) or c)  # Output: True
```
**Penjelasan:**
`(a and b)` dihitug dulu: `True and False` = `False`. Lalu `False or c` = `False or True` = `True`.

---

### 11. Gerbang AND dengan Input Berbeda
```python
a, b = False, False
print(a and b)  # Output: False
```
**Penjelasan:**
AND dengan kedua input `False` pasti menghasilkan `False`.

---

### 12. Gerbang OR dengan Input Berbeda
```python
a, b = False, False
print(a or b)  # Output: False
```
**Penjelasan:**
OR dengan kedua input `False` pasti menghasilkan `False`.

---

### 13. Gerbang NOT + AND
```python
a, b = True, False
print(not a and b)  # Output: False
```
**Penjelasan:**
`not a` = `False`, lalu `False and b` = `False and False` = `False`.

---

### 14. Gerbang NOT + OR
```python
a, b = True, False
print(not a or b)  # Output: False
```
**Penjelasan:**
`not a` = `False`, lalu `False or b` = `False or False` = `False`.

---

### 15. Gerbang XOR dengan 3 Input
```python
a, b, c = True, True, False
print((a != b) != c)  # Output: False
```
**Penjelasan:**
`a != b` = `False`, lalu `False != c` = `False != False` = `False`.

---

### 16. Gerbang NAND dengan 3 Input
```python
a, b, c = True, True, True
print(not (a and b and c))  # Output: False
```
**Penjelasan:**
`a and b and c` = `True`, lalu `not True` = `False`.

---

### 17. Gerbang NOR dengan 3 Input
```python
a, b, c = False, False, False
print(not (a or b or c))  # Output: True
```
**Penjelasan:**
`a or b or c` = `False`, lalu `not False` = `True`.

---

### 18. Gerbang XNOR dengan 3 Input
```python
a, b, c = True, True, True
print((a == b) == c)  # Output: True
```
**Penjelasan:**
`a == b` = `True`, lalu `True == c` = `True == True` = `True`.

---

### 19. Kombinasi Gerbang (AND + OR + NOT)
```python
a, b, c = True, False, True
print(not (a and b) or c)  # Output: True
```
**Penjelasan:**
`a and b` = `False`, lalu `not False` = `True`, kemudian `True or c` = `True or True` = `True`.

---

### 20. Gerbang Logika Kompleks
```python
a, b, c, d = True, False, True, False
print((a and b) or (c and d) or (a != b))  # Output: True
```
**Penjelasan:**
- `a and b` = `False`
- `c and d` = `False`
- `a != b` = `True`
- `False or False or True` = `True`

### 5. **List**
Digunakan untuk menyimpan kumpulan data yang dapat diubah (mutable).
**Contoh:**
```python
buah = ["apel", "jeruk", "mangga"]
print(buah)  # Output: ['apel', 'jeruk', 'mangga']
print(type(buah))  # Output: <class 'list'>
```

---

### 6. **Tuple**
Digunakan untuk menyimpan kumpulan data yang tidak dapat diubah (immutable).
**Contoh:**
```python
warna = ("merah", "hijau", "biru")
print(warna)  # Output: ('merah', 'hijau', 'biru')
print(type(warna))  # Output: <class 'tuple'>
```

---

### 7. **Dictionary (dict)**
Digunakan untuk menyimpan data dalam pasangan kunci-nilai (key-value).
**Contoh:**
```python
mahasiswa = {"nama": "Andi", "umur": 21}
print(mahasiswa)  # Output: {'nama': 'Andi', 'umur': 21}
print(type(mahasiswa))  # Output: <class 'dict'>
```

---

### 8. **Set**
Digunakan untuk menyimpan kumpulan data unik (tidak berurutan).
**Contoh:**
```python
hobi = {"membaca", "menulis", "berenang"}
print(hobi)  # Output: {'membaca', 'menulis', 'berenang'}
print(type(hobi))  # Output: <class 'set'>
```

### **Ringkasan Tabel Tipe Data Dasar**

| Tipe Data | Contoh Nilai | Keterangan |
|-----------|--------------|------------|
| int       | 10, -5       | Bilangan bulat |
| float     | 3.14, -0.5   | Bilangan desimal |
| str       | "Halo", 'A'  | Teks/karakter |
| bool      | True, False  | Nilai logika |
| list      | [1, 2, 3]    | Daftar data (mutable) |
| tuple     | (1, 2, 3)    | Daftar data (immutable) |
| dict      | {"a": 1}     | Pasangan kunci-nilai |
| set       | {1, 2, 3}    | Kumpulan data unik |

**Catatan:**
- Gunakan `type(nama_variabel)` untuk mengecek tipe data.
- Python bersifat dinamis, sehingga tipe data variabel dapat berubah sesuai nilai yang diberikan.
