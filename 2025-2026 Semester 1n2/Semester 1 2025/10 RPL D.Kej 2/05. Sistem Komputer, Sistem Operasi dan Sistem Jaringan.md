## ðŸ”¹ Pengertian Sistem Komputer
**Sistem komputer** adalah sekumpulan perangkat yang saling berhubungan dan bekerja sama untuk menjalankan proses pengolahan data sehingga menghasilkan informasi yang bermanfaat.  

âž¡ï¸ Sistem komputer terdiri dari **hardware (perangkat keras), software (perangkat lunak), dan brainware (pengguna)**.

---

## ðŸ”¹ Komponen Utama Sistem Komputer
1. **Hardware (Perangkat Keras)**
   - Komponen fisik komputer yang dapat dilihat dan disentuh.
   - Contoh: CPU, RAM, monitor, keyboard, mouse, harddisk, printer.

2. **Software (Perangkat Lunak)**
   - Program atau instruksi yang digunakan untuk menjalankan komputer.
   - Jenis software:
     - **Sistem Operasi** (Windows, Linux, macOS).
     - **Aplikasi** (Office, Browser, Multimedia).
     - **Utility** (Antivirus, Disk Management).

3. **Brainware (Pengguna)**
   - Orang yang menggunakan dan mengoperasikan komputer.
   - Contoh: operator, programmer, teknisi, administrator.

---

## ðŸ”¹ Fungsi Sistem Komputer
1. **Input** â†’ menerima data dari perangkat input.  
2. **Process** â†’ mengolah data menjadi informasi.  
3. **Output** â†’ menyajikan informasi ke pengguna.  
4. **Storage** â†’ menyimpan data dan informasi.  
5. **Control** â†’ mengatur dan mengendalikan semua proses kerja komputer.  

---

## ðŸ”¹ Struktur Sistem Komputer
1. **Unit Input**  
   Perangkat untuk memasukkan data.  
   âž¡ï¸ Contoh: keyboard, mouse, scanner, microphone.  

2. **Unit Proses (CPU)**  
   - **ALU (Arithmetic Logic Unit)** â†’ melakukan perhitungan aritmatika dan logika.  
   - **CU (Control Unit)** â†’ mengatur jalannya instruksi.  
   - **Register** â†’ penyimpanan kecil dan cepat untuk data sementara.  

3. **Unit Memori**  
   - **RAM (Random Access Memory)** â†’ penyimpanan sementara.  
   - **ROM (Read Only Memory)** â†’ menyimpan instruksi permanen.  
   - **Secondary Storage** â†’ harddisk, SSD, flashdisk.  

4. **Unit Output**  
   Perangkat untuk menampilkan hasil pengolahan data.  
   âž¡ï¸ Contoh: monitor, printer, speaker, proyektor.  

---

## ðŸ”¹ Jenis Sistem Komputer
1. **Supercomputer** â†’ komputer dengan kemampuan sangat tinggi, digunakan untuk penelitian, simulasi, dan analisis data besar.  
2. **Mainframe** â†’ digunakan oleh perusahaan besar untuk mengolah data transaksi dalam jumlah besar.  
3. **Minicomputer** â†’ lebih kecil dari mainframe, dipakai di laboratorium atau perusahaan menengah.  
4. **Personal Computer (PC)** â†’ komputer pribadi untuk keperluan sehari-hari.  
5. **Embedded System** â†’ sistem komputer yang tertanam dalam perangkat lain (contoh: smart TV, smartphone, mesin ATM).  

---

## ðŸ”¹ Perkembangan Sistem Komputer (Generasi Komputer)
1. **Generasi I (1940â€“1956)** â†’ tabung vakum.  
2. **Generasi II (1956â€“1963)** â†’ transistor.  
3. **Generasi III (1964â€“1971)** â†’ IC (Integrated Circuit).  
4. **Generasi IV (1971â€“sekarang)** â†’ mikroprosesor.  
5. **Generasi V (sekarang & masa depan)** â†’ AI (Artificial Intelligence), komputasi kuantum.

## ðŸ”¹ Contoh Penerapan Sistem Komputer

## 1ï¸âƒ£ Pengertian Sistem Operasi

### ðŸ§  Apa itu Sistem Operasi?
**Sistem Operasi (Operating System / OS)** adalah perangkat lunak utama yang mengatur seluruh kegiatan komputer, mulai dari mengelola perangkat keras (hardware), menjalankan aplikasi, hingga berinteraksi dengan pengguna.

> Tanpa sistem operasi, komputer hanyalah mesin tanpa kendali.

### ðŸ’¬ Analogi Sederhana
Bayangkan komputer seperti **sebuah pabrik**:
- **Pekerja dan mesin** = perangkat keras (hardware)
- **Manajer pabrik** = sistem operasi (OS)
- **Produk yang dihasilkan** = program atau aplikasi  
> Jadi, OS mengatur agar â€œpekerjaâ€ (hardware) dan â€œprogramâ€ (software) bisa bekerja bersama dengan baik.

---

## 2ï¸âƒ£ Fungsi Sistem Operasi

| Fungsi | Penjelasan Singkat |
|---------|--------------------|
| **Manajemen Perangkat Keras (Hardware Management)** | Mengatur penggunaan CPU, memori, harddisk, printer, dan perangkat lainnya. |
| **Manajemen File (File Management)** | Mengatur penyimpanan, pembacaan, dan penghapusan file. |
| **Manajemen Proses (Process Management)** | Mengatur program yang sedang dijalankan (multitasking). |
| **Manajemen Memori (Memory Management)** | Mengalokasikan dan mengelola ruang memori untuk setiap aplikasi. |
| **Manajemen Keamanan (Security Management)** | Melindungi data dan sistem dari akses tidak sah. |
| **Manajemen Pengguna (User Management)** | Mengatur hak akses pengguna ke sistem. |
| **Antarmuka Pengguna (User Interface)** | Menyediakan tampilan atau cara bagi pengguna berinteraksi dengan komputer (CLI/GUI). |

---

## 3ï¸âƒ£ Jenis-Jenis Sistem Operasi

### ðŸ§© Berdasarkan Antarmuka
| Jenis | Ciri-ciri | Contoh |
|--------|------------|---------|
| **GUI (Graphical User Interface)** | Berbasis tampilan grafis (ikon, jendela, klik). | Windows, macOS, Ubuntu Desktop |
| **CLI (Command Line Interface)** | Menggunakan perintah teks (terminal). | Linux Server, DOS, Mikrotik CLI |

### âš™ï¸ Berdasarkan Penggunaan
| Jenis | Deskripsi | Contoh |
|--------|------------|---------|
| **Sistem Operasi Komputer Pribadi** | Digunakan untuk PC/Laptop pengguna umum. | Windows 11, macOS, Ubuntu Desktop |
| **Sistem Operasi Server** | Digunakan untuk melayani jaringan dan pengguna lain. | Windows Server, Debian, CentOS |
| **Sistem Operasi Mobile** | Digunakan pada perangkat seluler. | Android, iOS |
| **Sistem Operasi Embedded** | Digunakan di perangkat khusus (IoT, router, mesin). | RouterOS, Android TV, Raspbian |

---

## 4ï¸âƒ£ Komponen Utama Sistem Operasi

| Komponen | Fungsi Utama |
|-----------|---------------|
| **Kernel** | Inti sistem operasi yang mengatur komunikasi antara hardware dan software. |
| **Shell** | Penghubung antara pengguna dan kernel (melalui CLI atau GUI). |
| **File System** | Struktur penyimpanan dan pengelolaan data pada media penyimpanan. |
| **Driver** | Program yang memungkinkan OS berkomunikasi dengan perangkat keras. |
| **User Interface** | Bagian tampilan yang digunakan oleh pengguna untuk berinteraksi. |

> ðŸ§  **Kernel = Otak Sistem Operasi**  
> Semua perintah dan proses dikendalikan oleh kernel sebelum dijalankan di hardware.

---

## 5ï¸âƒ£ Cara Kerja Sistem Operasi

1. **Booting**  
   Saat komputer dinyalakan, BIOS/UEFI menjalankan OS dari harddisk ke memori.
2. **Inisialisasi Sistem**  
   OS memuat driver, sistem file, dan konfigurasi awal.
3. **Menjalankan Proses**  
   OS mengatur program dan aplikasi agar berjalan bergantian (multitasking).
4. **Pengelolaan Sumber Daya**  
   OS membagi waktu CPU, memori, dan perangkat untuk setiap aplikasi.
5. **Interaksi Pengguna**  
   Pengguna berinteraksi melalui GUI (klik ikon) atau CLI (mengetik perintah).
6. **Shutdown / Restart**  
   OS menghentikan semua proses dan mematikan sistem dengan aman.
