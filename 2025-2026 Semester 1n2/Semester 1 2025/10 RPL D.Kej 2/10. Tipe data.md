## **1. Apa Itu Python?**
**Python** adalah bahasa pemrograman yang dirancang agar mudah dipahami dan digunakan, baik oleh pemula maupun programmer profesional. Sebagai bahasa **tingkat tinggi**, Python memungkinkan kita menulis kode dengan sintaksis sederhana dan mirip bahasa sehari-hari, sehingga tidak perlu memikirkan detail teknis yang rumit seperti pada bahasa pemrograman tingkat rendah. Python juga termasuk bahasa **interpreted**, artinya kode yang kita tulis dapat langsung dijalankan baris per baris tanpa perlu dikompilasi terlebih dahulu, sehingga proses pengembangan menjadi lebih cepat dan fleksibel. Selain itu, Python mendukung **multi-paradigma**, yang berarti kita bisa menulis program dengan berbagai gaya, seperti pemrograman prosedural (langkah demi langkah), pemrograman berorientasi objek (OOP) untuk membuat struktur kode yang terorganisir, maupun pemrograman fungsional yang fokus pada penggunaan fungsi-fungsi kecil. Dengan keunggulan ini, Python cocok digunakan untuk berbagai keperluan, mulai dari membuat website, menganalisis data, hingga mengembangkan kecerdasan buatan. Python dikenal karena sintaksisnya yang **sederhana**, **mudah dibaca**, dan **serbaguna**. Python banyak digunakan untuk:
- Pengembangan web (Django, Flask)
- Data Science & AI (NumPy, Pandas, TensorFlow)
- Otomatisasi & scripting
- Game & aplikasi desktop

## **4. Sintaksis Dasar Python**
### **A. Menampilkan Output**
```python
print("Halo, dunia!")
```
**Output:**
```

```

---

### 1. **Integer (int)**
Digunakan untuk menyimpan bilangan bulat (positif/negatif).
**Contoh:**
```python
umur = 20
print(umur)  # Output: 20
print(type(umur))  # Output: <class 'int'>
```

---

### 2. **Float (float)**
Digunakan untuk menyimpan bilangan desimal (pecahan).
**Contoh:**
```python
tinggi = 175.5
print(tinggi)  # Output: 175.5
print(type(tinggi))  # Output: <class 'float'>
```

---

### 3. **String (str)**
Digunakan untuk menyimpan teks atau karakter, diawali dan diakhiri dengan tanda petik (`'` atau `"`).
**Contoh:**
```python
nama = "Budi"
print(nama)  # Output: Budi
print(type(nama))  # Output: <class 'str'>
```

---

### 4. **Boolean (bool)**
Hanya memiliki dua nilai: `True` atau `False`.
**Contoh:**
```python
is_student = True
print(is_student)  # Output: True
print(type(is_student))  # Output: <class 'bool'>
```

---

### 5. **List**
Digunakan untuk menyimpan kumpulan data yang dapat diubah (mutable).
**Contoh:**
```python
buah = ["apel", "jeruk", "mangga"]
print(buah)  # Output: ['apel', 'jeruk', 'mangga']
print(type(buah))  # Output: <class 'list'>
```

---

### 6. **Tuple**
Digunakan untuk menyimpan kumpulan data yang tidak dapat diubah (immutable).
**Contoh:**
```python
warna = ("merah", "hijau", "biru")
print(warna)  # Output: ('merah', 'hijau', 'biru')
print(type(warna))  # Output: <class 'tuple'>
```

---

### 7. **Dictionary (dict)**
Digunakan untuk menyimpan data dalam pasangan kunci-nilai (key-value).
**Contoh:**
```python
mahasiswa = {"nama": "Andi", "umur": 21}
print(mahasiswa)  # Output: {'nama': 'Andi', 'umur': 21}
print(type(mahasiswa))  # Output: <class 'dict'>
```

---

### 8. **Set**
Digunakan untuk menyimpan kumpulan data unik (tidak berurutan).
**Contoh:**
```python
hobi = {"membaca", "menulis", "berenang"}
print(hobi)  # Output: {'membaca', 'menulis', 'berenang'}
print(type(hobi))  # Output: <class 'set'>
```

---

### **Ringkasan Tabel Tipe Data Dasar**

| Tipe Data | Contoh Nilai | Keterangan |
|-----------|--------------|------------|
| int       | 10, -5       | Bilangan bulat |
| float     | 3.14, -0.5   | Bilangan desimal |
| str       | "Halo", 'A'  | Teks/karakter |
| bool      | True, False  | Nilai logika |
| list      | [1, 2, 3]    | Daftar data (mutable) |
| tuple     | (1, 2, 3)    | Daftar data (immutable) |
| dict      | {"a": 1}     | Pasangan kunci-nilai |
| set       | {1, 2, 3}    | Kumpulan data unik |

---

**Catatan:**
- Gunakan `type(nama_variabel)` untuk mengecek tipe data.
- Python bersifat dinamis, sehingga tipe data variabel dapat berubah sesuai nilai yang diberikan.
